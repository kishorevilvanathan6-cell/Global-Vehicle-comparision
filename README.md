# Global Vehicle Sales Analysis Dashboard

## Overview
This Power BI dashboard analyzes global vehicle sales across Petrol, Diesel, and Electric Vehicle (EV) segments.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Key Features
- Total Sales KPI
- Minimum Sales KPI
- Maximum Sales KPI
- Country Ranking Analysis
- EV Subsidy Analysis
